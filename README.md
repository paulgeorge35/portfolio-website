## Getting Started

### Prerequisites

- Node.js 18.17 or later
- bun

### Installation

1. Clone the repo
```bash
git clone https://github.com/paulgeorge35/portfolio-website
```

2. Install packages
```bash
bun install
```

3. Run development server
```bash
bun run dev:bun
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
